# Salle Désilets

Landing page réalisée pour la Salle Désilets.

## Objectif
Mettre en valeur la programmation à travers une landing page claire, attrayante et responsive touten gardant en tête la clientèle cible plus âgé.

## Réalisation
- Intégration front end complète
- Intégration de l'identité visuelle du site et de la marque
- Structuration du contenu et des sections
- Adaptation responsive (desktop et mobile)
- Attention portée à la lisibilité, à l’ergonomie et à la hiérarchie visuelle

## Technologies utilisées
- HTML
- CSS
- JavaScript
